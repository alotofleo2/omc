# SCPictureBrowser
仿微信图片浏览

## Usage

---
``` bash
    SCPictureBrowser *browser = [SCPictureBrowser browserWithItems:_items
                                                       currentPage:_currentPage
                                              numberOfPrefetchURLs:2];
    [browser show];
```
